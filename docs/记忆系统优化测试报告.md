## 已知限制

1. **因果链 LLM 识别未接入**：`caused_by_id` 字段和 `retrieve_causal_chain` 函数已就绪，但反思 prompt 尚未输出因果关联。需后续按需接入。
2. **去重基于字符集交集**：对短文本（<10 字）可能误判。当前阈值 0.65 经测试对中等长度文本有效。
3. **关系权重仅在检索时生效**：关系数据积累后，高信任角色的记忆会被优先检索，但低信任角色的记忆不会被完全屏蔽。
